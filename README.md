# DSA Visualizer Pro

A **React-based interactive Data Structures & Algorithms Visualizer** to help learners understand BFS, DFS, sorting algorithms, stacks, queues, and trees.

## Features
- **Graph Traversals:** BFS and DFS with auto and step-by-step animation  
- **Sorting Algorithms:** Bubble Sort, Merge Sort, Quick Sort  
- **Stack & Queue:** Interactive push/pop and enqueue/dequeue visualizations  
- **Trees & MST:** Binary tree and Minimum Spanning Tree visualization  
- **Animation Control:** Adjustable speed for all algorithms  
- Fully responsive and user-friendly UI

## Technologies
- React.js  
- JavaScript (ES6)  
- CSS3
